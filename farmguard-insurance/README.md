# 🌾 FarmGuard Insurance

Blockchain-based parametric crop insurance on Solana.

## Features
- **Zero-Claim Filing**: Payouts trigger automatically via weather oracles.
- **Fast Liquidity**: Funds arrive in farmer wallets within 48 hours.
- **Satellite Auditing**: Uses NDVI health scores for crop failure verification.
